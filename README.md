# 🚀 ModelVerse AI

### 🔍 Explore • Compare • Understand Machine Learning in One Platform

---

## 🌟 Project Overview

**ModelVerse AI** is a complete **Machine Learning Dashboard** that demonstrates all major ML concepts in a **single application**.

This project covers:

* 📈 Regression (Predict numerical values)
* 📊 Classification (Predict categories)
* 🤖 Ensemble Learning (Advanced models)
* 🔍 Clustering (Unsupervised learning)

👉 Built for:

* Academic submission
* Resume project
* Practical ML understanding
